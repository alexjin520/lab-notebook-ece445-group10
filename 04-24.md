## Overview

During these three weeks, we focused on assembling the physical prototype and verifying that each subsystem works correctly. The main goal was to move from individual component testing toward a more integrated head-and-waist module design.

## Hardware Assembly

We purchased the required wires, tapes, and basic assembly materials needed for connecting and securing the prototype components. Using these materials, we completed the physical assembly of both the **waist module** and the **head module**.

The waist module and head module were arranged according to our system design so that the sensors, microcontroller, power connections, and feedback components could be tested in a more realistic wearable configuration.

## Subsystem Testing

After assembling the modules, we tested each subsystem individually to make sure it was functioning properly before combining everything into the full system.

The subsystems we checked include:

- Power delivery and wiring connections
- Head module sensor connections
- Waist module sensor connections
- Haptic feedback response
- Basic communication between the control unit and connected components
- Physical stability of the assembled modules

Overall, the tested subsystems were able to operate as expected. The wiring and module layout were also adjusted during testing to improve reliability and reduce loose connections.

## Current Progress

By the end of these three weeks, we completed the following:

- Purchased wires, tapes, and assembly materials
- Finished assembling the waist module
- Finished assembling the head module
- Verified that the main subsystems are functioning normally
- Improved the physical organization of the prototype wiring
- Confirmed that the current hardware layout is suitable for further integration testing
